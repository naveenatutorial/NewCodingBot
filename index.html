My Projects

project 1

Smart Attendance System This project is a Smart Attendance System that combines both hardware and software components. The system uses an RFID module and Arduino (hardware) to scan student ID cards and collect attendance data. The collected data is then sent to a web application developed using HTML, CSS, and JavaScript (software), where attendance records are stored and displayed. This project helps automate the attendance process, reduces manual work, and improves accuracy. It also provides a simple dashboard for viewing student attendance details.

Project 2

Smart Home Automation System This project is a Smart Home Automation System that integrates both hardware and software components. The system uses an Arduino/NodeMCU and relay modules (hardware) to control electrical appliances like lights and fans. A web-based application developed using HTML, CSS, and JavaScript (software) allows users to control the devices remotely through a simple dashboard. This project improves convenience, saves energy, and enhances home safety by enabling users to monitor and control appliances from anywhere.
